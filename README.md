## 프로젝트 소개

코어뷰는 Code Review라는 뜻으로, 코드를 입력하고 숙련된 개발자들에게 자신의 코드를 피드백 받는 사이트입니다. 보통의 사이트는 각 줄 마다 피드백을 줄 수 없어서, 긴 코드의 경우 피드백을 주기 어려움이 있다는 아쉬움을 해결하고자 만들어졌습니다.
